# whitespaces

**Fix all the whitespaces!!**

## Installation
```
npm install whitespaces
```

## Usage

```
whitespaces [FILES]
```

* Fixes trailing whitespaces
* Fixes whitespaces in blank lines

If you like this project, please watch this and follow me.
